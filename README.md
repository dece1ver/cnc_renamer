# CNC Renamer
Программа переименовывает файлы управляющих программ по названию самой управляющей программы.

Поддерживаемые СЧПУ:
* Fanuc 0i           [ O0001(НАЗВАНИЕ) ]
* Fanuc 0i-*F        [ <НАЗВАНИЕ> ]
* Mazatrol Smart     [ .PBG | .PBD ]
* Sinumerik 840D sl  [ MSG (\"НАЗВАНИЕ\") ]
* Hiedenhain         [ BEGIN PGM НАЗВАНИЕ MM ]
